# kein-backend-kein-problem

Folien für den Vortrag "Kein Backend, kein Problem! Wie bringen wir Inhalte lange ins Netz?" - Ringvorlesung Uni Bern Aktuelle digitale Editionen höfischer Literatur des Mittelalters Mai 2026.

## Installation and usage

Requirements: node and npm

- use `npm install` to install the package
- then use `npm run open` to start the presentation

If you want to use the CLI-options of eloc, use `npm exec -- eloc`.

Or simlply `eloc kein-backend-kein-problem.md`

## Commands

| presentation                              | command                                                             |
| ----------------------------------------- | ------------------------------------------------------------------- |
| [kein-backend-kein-problem](kein-backend-kein-problem.md) | `npm exec -- eloc open kein-backend-kein-problem.md --css "style.css"` |
